   - 输入WiFi密码
   - 在"MusicXML URL"字段输入乐谱文件的完整URL

6. 点击"Save"保存

## 翻页操作

### 方法1：陀螺仪晃动

**向右晃动（下一页）：**
- 握住设备，快速向右晃动
- 屏幕右下角会短暂显示绿色提示：`Next >>`
- 自动跳转到下一页

**向左晃动（上一页）：**
- 握住设备，快速向左晃动
- 屏幕左下角会短暂显示蓝色提示：`<< Prev`
- 自动跳转到上一页

**注意事项：**
- 晃动需要有一定力度才能触发
- 两次晃动之间有800ms的防抖时间
- 如果太灵敏或不够灵敏，可以调整代码中的`shakeThreshold`参数

### 方法2：触摸屏

**点击右侧（下一页）：**
- 点击屏幕右半部分（X > 160）
- 点击位置会显示绿色圆圈反馈
- 自动跳转到下一页

**点击左侧（上一页）：**
- 点击屏幕左半部分（X ≤ 160）
- 点击位置会显示蓝色圆圈反馈
- 自动跳转到上一页

## 重新配置

如果需要更改WiFi或MusicXML URL：

### 方法1：清除配置（推荐）
1. 通过串口连接设备
2. 在串口监视器中输入特定命令（需要在代码中实现）
3. 或者修改代码添加重置按钮功能

### 方法2：重新烧录
1. 重新上传程序到设备
2. 设备会自动进入配置模式

### 方法3：代码修改
在`src/main.cpp`中添加以下代码来清除配置：
```cpp
// 在setup()开始处添加
if (M5.BtnA.wasPressed()) {
    preferences.clear();
    wifiManager.resetSettings();
    ESP.restart();
}
```

## 高级功能

### 调整陀螺仪灵敏度

编辑`src/GyroPageTurner.cpp`：
```cpp
GyroPageTurner::GyroPageTurner() 
    : lastAccelX(0), 
      shakeThreshold(1.5),  // 调整这个值
      // 增大 = 需要更大力度晃动
      // 减小 = 更容易触发
```

### 调整防抖时间

```cpp
GyroPageTurner::GyroPageTurner() 
    : lastAccelX(0), 
      shakeThreshold(1.5), 
      lastShakeTime(0), 
      shakeDebounceMs(800),  // 调整这个值（毫秒）
      // 增大 = 翻页间隔更长
      // 减小 = 可以更快连续翻页
```

### 调整每页显示的小节数

编辑`src/StaffRenderer.cpp`：
```cpp
StaffRenderer::StaffRenderer() 
    : measures(nullptr), 
      measuresPerPage(4)  // 调整这个值
      // 增大 = 每页显示更多小节（可能拥挤）
      // 减小 = 每页显示更少小节（更清晰）
```

## 故障排除

| 问题 | 可能原因 | 解决方法 |
|------|---------|---------|
| 无法创建热点 | ESP32 WiFi模块问题 | 重启设备，检查硬件 |
| 配置页面无法打开 | 浏览器缓存 | 清除缓存或使用无痕模式 |
| 下载很慢 | 网络速度慢 | 等待或使用更快的网络 |
| 乐谱显示不完整 | 文件过大或格式复杂 | 简化MusicXML文件 |
| 陀螺仪不响应 | IMU初始化失败 | 重启设备 |
| 触摸屏不响应 | 触摸校准问题 | 重启设备 |

## 技巧和建议

1. **最佳MusicXML文件大小**：建议小于100KB，确保快速下载和解析
2. **WiFi信号**：保持设备在WiFi信号良好的区域使用
3. **电池续航**：WiFi和屏幕常亮会消耗电量，建议连接电源使用
4. **乐谱准备**：使用MuseScore等软件导出标准的MusicXML格式
5. **测试文件**：先用简单的乐谱测试，确认功能正常后再使用复杂乐谱

## 示例MusicXML托管

你可以使用以下方式托管MusicXML文件：

1. **GitHub Pages**：将XML文件上传到GitHub仓库，启用Pages
2. **个人服务器**：使用Apache/Nginx托管
3. **云存储**：使用支持直链的云存储服务
4. **在线乐谱库**：使用公开的MusicXML资源

确保URL可以直接访问，不需要登录或验证。
